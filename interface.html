First Arriving Media Sourcer

Directions:

  • Click on "Copy Image Source"
  • Navigate to the image or video you want to check the source for
  • Once you click on the media, the source will be copied to your clipboard and a new page will open up to test if it can be iframed
  • If it can be iframed, paste the src string into the dashboard you are working on
  • If a new tab does not open, a media source was not clicked or a src was not publicly available
